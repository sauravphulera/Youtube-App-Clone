#### Youtube Clone

This projects contains the Frontend for an MVP clone of youtube app in react. Clone the application and run 
npm i & npm start
then you are good to go.

## Store

# Redux Toolkit
- Used  For caching search results in auto complete
- Used for Chat Updated and Live chat data
- Used for toggling sidebar menu from anywhere
